# <a name="mssqlserver12302"></a>MSSQLSERVER_12302
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|SQL Server|  
|イベント ID|12302|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|メッセージ テキスト|PRIMARY KEY 制約の一部である列の更新は '*construct*' でサポートされていません。|
