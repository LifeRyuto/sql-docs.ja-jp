# <a name="sqlnumparams"></a>SQLNumParams
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  データベース エンジンの機能強化を最初[!INCLUDE[ssSQL11](../../includes/sssql11-md.md)]に行うと、SQLNumParams は期待される結果をより正確に説明できます。 これらのより正確な結果は、以前のバージョンの[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]SQLNumParams によって返される値とは異なる場合があります。  
  
 詳細については、「[メタデータの検出](../../relational-databases/native-client/features/metadata-discovery.md)」を参照してください。
