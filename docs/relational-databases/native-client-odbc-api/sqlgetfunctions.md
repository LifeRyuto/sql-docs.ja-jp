# <a name="sqlgetfunctions"></a>SQLGetFunctions
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  ネイティブ[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]クライアント ODBC ドライバー インターフェイスは、ODBC (ISO、オープン グループ、および ODBC レベル 2) の 3 つの定義済みの準拠レベルに準拠しています。 **SQLGetFunctions は**、ネイティブ クライアント ODBC[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]ドライバーのすべての ODBC 関数がサポートされていることを報告します。
