# <a name="servernetworkprotocol-class"></a>ServerNetworkProtocol クラス
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [ServerNetworkProtocol クラス](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md)ネットワークのインスタンスにインストールされているプロトコルを表します[!INCLUDE[msCoName](../../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]をネットワーク経由でクライアントと通信できるようにします。  
  
 [ServerNetworkProtocol クラス](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md)を実行できます。  
  
-   参照されているサーバー ネットワーク プロトコルの名前、バージョン、.dll ファイル、プロパティ数、表示名の確認。  
  
-   ネットワーク プロトコルのプロパティおよび IP アドレスの参照。  
  
-   サーバー ネットワーク プロトコルの有効化および無効化。
