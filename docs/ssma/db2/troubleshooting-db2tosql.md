# <a name="troubleshooting-db2tosql"></a>トラブルシューティング (DB2ToSQL)
コンソールのスクリプトまたはコマンドを実行する移行の問題の途中に実行することは珍しくありません。  
  
修正可能であるエラーの関連する可能性があります。  
  
1.  関数  
  
2.  手順  
  
3.  表示  
  
4.  トリガー  
  
5.  データ型を変更します。  
  
6.  プロジェクトの設定を変更します。  
  
SSMA ユーザー インターフェイスを使用して、コンソールを使って作成されたプロジェクト ファイルを開き、問題を修正できます。
