# <a name="internet-explorer-error-codes-for-remote-data-service"></a>リモート データ サービスのインターネット エクスプローラ エラー コード
次の表は、リモート データ サービスの使用状況に関連する Microsoft ® Internet Explorer エラー コードの一覧です。 下位 2 バイトの正の 10 進変換、エラー コード全体の負の 10 進変換、および 16 進値が示されます。

|インターネット エクスプローラ (Wininet) エラー|Number|説明|
|------------------------------------------|------------|-----------------|
|**IDS_WinInet_CantConnect**|8195 -2146820093 0x800A2003|インターネット クライアント エラー : サーバーに接続できません。|
|**IDS_WinInet_接続リセット**|12031 -2146816257 0x800A2EFF|インターネット クライアント エラー: 接続のリセット。|
|**IDS_WinInet_Error**|8193 -2146820095 0x800A2001|インターネット クライアント エラーです。|
|**IDS_WinInet_無効なサーバー応答**|8430 -2146819858 0x800A20EE|インターネット クライアント エラー: サーバーの応答が無効です。|
|**IDS_WinInet_SSL ポスト制限**|8196 -2146820092 0x800A2004|インターネット クライアント エラー: TLS/SSL エラー (場合によっては 32K データのアップロード制限)。|
|**IDS_WinInet_Timeout**|8194 -2146820094 0x800A2002|インターネット クライアント エラー: 要求のタイムアウトです。|
