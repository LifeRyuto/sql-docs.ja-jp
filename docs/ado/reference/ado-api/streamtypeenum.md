# <a name="streamtypeenum"></a>StreamTypeEnum
[ストリーム](../../../ado/reference/ado-api/stream-object-ado.md)オブジェクトに格納されているデータの型を指定します。  
  
|常時|値|[説明]|  
|--------------|-----------|-----------------|  
|**adTypeBinary**|1 で保護されたプロセスとして起動されました|バイナリデータを示します。|  
|**adTypeText**|2|既定。 [Charset](../../../ado/reference/ado-api/charset-property-ado.md)によって指定された文字セットに含まれるテキストデータを示します。|  
  
## <a name="adowfc-equivalent"></a>同等の ADO/WFC  
 これらの定数には、対応する ADO/WFC がありません。
