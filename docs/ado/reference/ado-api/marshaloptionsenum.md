# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
サーバーに返すレコードを指定します。  
  
|常時|値|[説明]|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|既定。 サーバーにすべての行を返します。|  
|**adMarshalModifiedOnly**|1 で保護されたプロセスとして起動されました|サーバーに変更された行のみを返します。|
