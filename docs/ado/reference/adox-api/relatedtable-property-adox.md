# <a name="relatedtable-property-adox"></a>RelatedTable プロパティ (ADOX)
関連テーブルの名前を示します。  
  
## <a name="settings-and-return-values"></a>設定と戻り値  
 設定し、返します、**文字列**値。  
  
## <a name="remarks"></a>コメント  
 既定値は空の文字列 ("") です。  
  
 キーが外部キーでは、次に設定されている場合**RelatedTable**キーを含むテーブルの名前を指定します。
