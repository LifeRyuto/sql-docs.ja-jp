# <a name="formattedvalue-property-ado-md"></a>FormattedValue プロパティ (ADO MD)
書式設定された表示を示します、[セル](../../../ado/reference/ado-md-api/cell-object-ado-md.md)値。  
  
## <a name="return-values"></a>戻り値  
 返します、**文字列**は読み取り専用であるとします。  
  
## <a name="remarks"></a>コメント  
 使用して、 **FormattedValue**プロパティの表示の書式設定の値を取得する、[値](../../../ado/reference/ado-md-api/value-property-ado-md.md)のプロパティを[セル](../../../ado/reference/ado-md-api/cell-object-ado-md.md)オブジェクト。 たとえば、セルの値が 1056.87 し、この値をドル金額を表す**FormattedValue** $1,056.87 になります。
