# <a name="move-or-delete-an-item-report-manager"></a>移動または削除するアイテム (レポート マネージャー)
  レポートおよびレポート サーバーにパブリッシュするレポート関連アイテムは、フォルダーに格納されます。 別のフォルダーにアイテムを移動することができ、それらの項目への参照は、レポート サーバーによって自動的に保持されます。 項目を削除する前に、その他の項目がそれに依存しているかどうかを検討してください。  
  
## <a name="move-an-item"></a>項目を移動する  
 レポート サーバー アイテムは、レポート サーバーのフォルダー階層内の別のフォルダーの場所に移動できます。 アイテムを移動するときに (セキュリティ設定を含む) すべてのプロパティが項目に新しい場所に移動します。 フォルダーを移動すると、フォルダー内のすべての項目はそれに移動します。  
  
 レポート マネージャーで、移動できるアイテムはフォルダー階層で示されます。 次の表には、移動可能な各アイテムのアイコンが表示されます。  
  
|アイコン|移動可能なアイテム|  
|----------|-------------------|  
|![レポート アイコン](../media/hlp-16doc.gif "レポート アイコン")|レポート|  
|![リンク レポート アイコン](../media/hlp-16linked.gif "リンク レポート アイコン")|リンク レポート|  
|![フォルダー アイコン](../media/hlp-16folder.gif "フォルダー アイコン")|フォルダー|  
|![汎用リソース アイコン](../media/hlp-16file.gif "汎用リソース アイコン")|汎用リソース|  
|![共有データ ソースのアイコン](../media/hlp-16datasource.png "共有データ ソースのアイコン")|[共有データ ソース]|  
||共有データセット|  
  
 使用するすべての項目を移動することができます。 サブスクリプションまたはレポート履歴など、レポートに関連付けられている項目を移動することはできません。 これらの項目は、関連付けられたレポートと共に移動します。 同様に、フォルダー階層の外部に存在すると、共有スケジュールなどの項目を移動することはできません。 そのアクセス許可がない場合、項目を移動することはできません。 アイテムを移動するためのアクセス許可を伝達するは、当該アイテムのロールの割り当てで、次のタスクが選択されている場合。"管理 reports、"「管理モデル」、「フォルダーを管理」と「データ ソースの管理」。  
  
#### <a name="to-move-an-item-from-within-the-contents-page"></a>[コンテンツ] ページ内からアイテムを移動するには  
  
1.  開始 [レポート マネージャー &#40;SSRS ネイティブ モード&#41;]./[レポートのマネージャーの ssrs-ネイティブ-mode.md) で変更します。  
  
2.  レポート マネージャーに移動し、**内容**ページ、および移動するアイテムを探します。  
  
3.  アイテムの上にマウス ポインターを移動し、下矢印をクリックします。  
  
4.  ドロップダウン メニューでクリックして**移動**します。  
  
5.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
6.  **場所**アイテムを移動するフォルダーを指定します。 フォルダーの完全修飾名を入力したり、ツリー コントロールを使用して、フォルダーに移動します。  
  
7.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
 移動オブジェクトを移動する代わりに、**プロパティ**、 をクリックし、**移動**ページの上部にあります。  
  
## <a name="delete-an-item"></a>項目を削除する  
 項目を削除する前に、かどうかは、他のアイテムによって使用を決定します。 たとえば、共有データ ソースを削除すると、そのデータ ソースを使用するレポートおよびモデルは実行できなくなります。 レポートを削除する場合は、サブスクリプションおよびそのレポートに関連付けられたレポート履歴も削除されます。 項目の依存アイテムを検索する、次を参照してください [依存アイテム] ページ&#40;レポート マネージャー&#41;]../[依存-項目のページのレポート-manager.md) で変更します。  
  
#### <a name="to-delete-a-report-or-item"></a>レポートまたはアイテムを削除するには  
  
1.  開始 [レポート マネージャー &#40;SSRS ネイティブ モード&#41;]./[レポートのマネージャーの ssrs-ネイティブ-mode.md) で変更します。  
  
2.  レポート マネージャーに移動し、**内容**ページ、および削除するアイテムを探します。  
  
3.  アイテムの上にマウス ポインターを移動し、下矢印をクリックします。  
  
4.  ドロップダウン メニューでクリックして**削除**します。  
  
5.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]
