# <a name="cube-selection-dialog-box-report-builder"></a>[キューブの選択] ダイアログ ボックス (レポート ビルダー)
  
  **[キューブの選択]** ダイアログ ボックスを使用すると、レポートのデータ ソースとして別のキューブを選択できます。 別のキューブを選択することにより、クエリは無効になります。
