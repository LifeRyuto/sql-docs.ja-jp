# <a name="after-upgrade-full-text-search-will-not-allow-predicates-in-output-into-expression"></a>アップグレード後、フルテキスト検索では、OUTPUT INTO 式で述語を使用できません。
  データベースの互換性レベルが 100 以上に設定されている場合、OUTPUT 句でフルテキスト述語を使用することはできません。
