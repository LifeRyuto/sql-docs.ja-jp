# <a name="reporting-services-parameters"></a>Reporting Services パラメーター
  を分析[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]するには、のインスタンスを[!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]指定し、そのインスタンスへの接続方法に関する情報を提供する必要があります。  
  
## <a name="options"></a>オプション  
 **[Reporting Services インスタンス名]**  
 一覧から [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] のインスタンスを選択します。
