# <a name="completing-the-wizard-linked-object-wizard"></a>[ウィザードの完了] (リンク オブジェクト ウィザード)
  
  **[ウィザードの完了]** ページを使用すると、ウィザードの前のページの指定を確認できます。  
  
## <a name="options"></a>オプション  
 **プレビュー**  
 リンクまたはインポート用に選択されたオブジェクトを表示します。
