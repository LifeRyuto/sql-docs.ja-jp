# <a name="completing-the-wizard-business-intelligence-wizard"></a>[ウィザードの完了] (ビジネス インテリジェンス ウィザード)
  **[ウィザードの完了]** ページを使用すると、ビジネス インテリジェンス ウィザードが完了するときに行われる変更を確認できます。  
  
## <a name="options"></a>オプション  
 **[変更点]**  
 プロジェクト内の[!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]オブジェクトに対して行われた変更を表示します。 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]
