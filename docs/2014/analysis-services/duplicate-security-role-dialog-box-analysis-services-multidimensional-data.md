# <a name="analysis-services-duplicate-security-role-dialog-box-analysis-services---multidimensional-data"></a>[Analysis Services のセキュリティ ロールの複製] ダイアログ ボックス (Analysis Services - 多次元データ)
  の [ **Analysis Services のセキュリティロールの\<複製-データベース名\<>-ロール名>からコピー** ] [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)]ダイアログボックスを使用して、 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]データベース内の既存のロールを複製します。 **[Analysis Services のセキュリティ ロールの複製]** ダイアログ ボックスを表示するには、オブジェクト エクスプローラーでロールを右クリックして **[複製]** をクリックします。  
  
## <a name="options"></a>オプション  
  
|用語|定義|  
|----------|----------------|  
|**[新しいロール名]**|オブジェクト エクスプローラーで選択したロールと同じメンバーと権限で作成される、新しいロールの名前を入力します。|
