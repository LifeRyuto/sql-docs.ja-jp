# <a name="delete-operator"></a>Delete 操作
  このページを使用すると、オペレーターを削除できます。  
  
## <a name="options"></a>オプション  
 **[削除されるオブジェクト]**  
 削除されるオペレーターを表示します。  
  
 **再割り当て**  
 削除されるオペレーターに通知を再割り当てします。  
  
 **Properties**  
 通知を再割り当てする対象オペレーターのプロパティを表示します。
