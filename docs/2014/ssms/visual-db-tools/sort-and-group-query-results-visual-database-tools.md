# <a name="sort-and-group-query-results-visual-database-tools"></a>クエリ結果の並べ替えおよびグループ化 (Visual Database Tools)
  基になるデータの行グループ全体に結果の各行が対応するようなクエリ結果を作成することができます。  
  
 このようなクエリの作成の詳細については、次の表の各トピックを参照してください。  
  
## <a name="in-this-section"></a>このセクションの内容  
 [Visual Database Tools &#40;行の並べ替え&#41;](visual-database-tools.md)  
 行を並べ替える各種の方法、およびそれらを使用する目的について説明します。  
  
 [Visual Database Tools &#40;行のグループを折りたたむ&#41;](collapse-groups-of-rows-visual-database-tools.md)  
 計算や複製の削除など、行を折りたたむ各種の方法について説明します。  
  
 [Visual Database Tools を使用した ORDER BY &#40;の並べ替え&#41;](sort-with-order-by-visual-database-tools.md)  
 特定の順序で結果を返す方法を説明します。  
  
 [Visual Database Tools&#41;&#40;昇順または降順で並べ替える](sort-in-ascending-or-descending-order-visual-database-tools.md)  
 並べ替えの方向を変更、または設定する手順を説明します。  
  
 [Visual Database Tools &#40;クエリ内の複数の列を並べ替える&#41;](sort-multiple-columns-in-queries-visual-database-tools.md)  
 複数列に対して結果セットの順序を設定する手順を説明します。  
  
 [クエリ結果の行をグループ化するには &#40;Visual Database Tools&#41;](group-rows-in-query-results-visual-database-tools.md)  
 データをグループ化して集計情報のサブセットを作成する手順を説明します。  
  
 [Visual Database Tools &#40;グループの条件を指定&#41;](specify-conditions-for-groups-visual-database-tools.md)  
 行のグループに適用する検索条件を作成する手順を説明します。  
  
 [Visual Database Tools&#41;&#40;出力列の順序を変更する](reorder-output-columns-visual-database-tools.md)  
 現在の並べ替え設定を変更する手順を説明します。
