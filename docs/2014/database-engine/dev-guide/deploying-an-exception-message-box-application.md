# <a name="deploying-an-exception-message-box-application"></a>例外メッセージ ボックス アプリケーションの配置
  例外メッセージボックスはによっ[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]てインストールされ、カスタム Windows アプリケーションでの使用がサポートされているため、例外処理を改善できます。 例外メッセージボックスは、を除く[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssEW](../../includes/ssew-md.md)]のすべてのエディションでインストールされるため、SDK を含む[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]クライアントコンポーネントがインストールされているコンピューターでは、追加の構成なしで使用できます。
