Linux マシンが前のセクションで使われたオンライン リポジトリにアクセスできない場合は、パッケージ ファイルを直接ダウンロードできます。 これらのパッケージが、Microsoft リポジトリにある[ https://packages.microsoft.com](https://packages.microsoft.com)します。

> [!TIP]
> 前のセクションの手順で正常にインストールした場合は、次のパッケージをダウンロードしたり手動でインストールする必要はありません。 これは、オフライン シナリオのみです。
