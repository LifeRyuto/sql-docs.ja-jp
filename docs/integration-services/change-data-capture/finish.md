# <a name="finish"></a>[完了] 

[!INCLUDE[ssis-appliesto](../../includes/ssis-appliesto-ssvrpluslinux-asdb-asdw-xxx.md)]


  **[完了]** ページには、ウィザードの処理がまとめて表示されます。 **[完了]** をクリックしてウィザードを閉じると、CDC インスタンスの作成ウィザードで定義したすべてのテーブルが生成されます。
