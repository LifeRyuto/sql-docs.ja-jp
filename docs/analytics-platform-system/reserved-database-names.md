# <a name="reserved-database-names-in-parallel-data-warehouse"></a>Parallel Data Warehouse で予約されているデータベース名
予約されているデータベース名では、SQL Server PDW のユーザー データベースに割り当てることができない名前です。  
  
予約されているデータベース名のリスト:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
