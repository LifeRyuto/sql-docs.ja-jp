# <a name="addbatch-method-"></a>addBatch () メソッド
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) オブジェクトのコマンド バッチにパラメーターのセットを追加します。  
  
## <a name="syntax"></a>構文  
  
```  
  
public final void addBatch()  
```  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 この addBatch メソッドは、java.sql.PreparedStatement インターフェイスの addBatch メソッドによって指定されます。
