# <a name="createblob-method-sqlserverconnection"></a>createBlob メソッド (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  データが含まれていない Blob オブジェクトを作成します。  
  
## <a name="syntax"></a>構文  
  
```  
  
public java.sql.Blob createBlob()  
```  
  
## <a name="return-value"></a>戻り値  
 Blob オブジェクト。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 この createBlob メソッドは、java.sql.Connection インターフェイスの createBlob メソッドによって指定されます。  
  
 このメソッドは置換の必要性[SQLServerBlob コンス トラクター &#40;SQLServerConnection、バイト&#41;](../../../connect/jdbc/reference/sqlserverblob-constructor-sqlserverconnection-byte.md)します。
