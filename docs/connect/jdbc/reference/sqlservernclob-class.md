# <a name="sqlservernclob-class"></a>SQLServerNClob クラス
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

 **パッケージ:** com.microsoft.sqlserver.jdbc  
  
 **拡張:** SQLServerClob  
  
 **実装:** java.sql.NClob

```  
  
public class SQLServerNClob
```  
