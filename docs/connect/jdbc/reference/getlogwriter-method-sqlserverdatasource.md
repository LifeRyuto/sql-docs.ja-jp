# <a name="getlogwriter-method-sqlserverdatasource"></a>getLogWriter メソッド (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  このメソッドは、内部でのみ使用されます。 ログ記録の詳細については、次を参照してください。 [Tracing Driver Operation](../../../connect/jdbc/tracing-driver-operation.md)します。  
  
## <a name="syntax"></a>構文  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>戻り値  
 PrintWriter オブジェクト。  
  
## <a name="remarks"></a>Remarks  
 この getLogWriter メソッドは、javax.sql.DataSource インターフェイスで getLogWriter メソッドによって指定されます。
