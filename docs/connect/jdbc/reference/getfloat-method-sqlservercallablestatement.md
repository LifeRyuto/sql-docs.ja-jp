# <a name="getfloat-method-sqlservercallablestatement"></a>getFloat メソッド (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  指定されたパラメーターの値を Java プログラミング言語の **float** として取得します。  
  
## <a name="overload-list"></a>オーバーロードの一覧  
  
|[オブジェクト名]|[説明]|  
|----------|-----------------|  
|[getFloat (int)](../../../connect/jdbc/reference/getfloat-method-int.md)|パラメーターに渡されたインデックスを使用して、指定されたパラメーターの値を Java プログラミング言語の **float** として取得します。|  
|[getFloat (java.lang.String)](../../../connect/jdbc/reference/getfloat-method-java-lang-string.md)|パラメーターに渡された名前を使用して、指定されたパラメーターの値を Java プログラミング言語の **float** として取得します。|
