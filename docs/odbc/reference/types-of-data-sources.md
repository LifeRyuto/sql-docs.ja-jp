# <a name="types-of-data-sources"></a>データ ソースの種類
データソースには、マシンデータソースとファイルデータソースの2種類があります。 どちらにもデータのソースに関する類似の情報が含まれていますが、この情報の格納方法は異なります。 これらの違いにより、これらは多少異なる方法で使用されます。
