**適合 性**

 **まとめ**  
 ODBC 3 で。*x、ODBC*2.0 関数**SQLGetStmtOption が** **SQLGetStmtAttr**に置き換えられました。 詳細については、「 [SQLGetStmtAttr](../../../odbc/reference/syntax/sqlgetstmtattr-function.md)」を参照してください。
