# <a name="sqlnumresultcols-function"></a>SQLNumResultCols 関数
**準拠**  
 バージョンが導入されました。ODBC 1.0 規格に準拠します。ISO 92  
  
 **まとめ**  
 **SQLNumResultCols**結果セット内の列の数を返します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
  
SQLRETURN SQLNumResultCols(  
     SQLHSTMT        StatementHandle,  
     SQLSMALLINT *   ColumnCountPtr);  
```  
  
## <a name="arguments"></a>引数  
 *StatementHandle*  
 [入力]ステートメント ハンドルです。  
  
 *ColumnCountPtr*  
 [出力]結果の列の数を返すバッファーへのポインターを設定します。 この数には、バインドされたブックマーク列は含まれません。  
  
## <a name="returns"></a>戻り値  
 SQL_SUCCESS、SQL_SUCCESS_WITH_INFO、SQL_STILL_EXECUTING、SQL_ERROR、または SQL_INVALID_HANDLE します。
