# <a name="numeric-literals"></a>数値リテラル
数値リテラルは、数値データ値が文字列に格納されている場合に使用されます。 数値 SQL データをSQL_C_CHAR文字列に変換したり、数値 C データを SQL_CHAR またはSQL_VARCHAR文字列に変換したりするために、数値リテラル構文を使用して、ターゲットに格納される内容を指定します。 SQL_C_CHAR文字列として格納された数値を数値 SQL データに変換する場合、または SQL_CHAR文字列として格納されている数値を数値 C データに変換する場合、この構文を使用してソースに格納されているデータを検証します。  
  
 数値リテラルは、「付録 C: SQL 文法」の[「数値リテラル構文](../../../odbc/reference/appendixes/numeric-literal-syntax.md)」で定義されている構文に準拠する必要があります。
