# <a name="literals-in-odbc"></a>ODBC のリテラル
以下のセクションの構文は、ODBC の間隔リテラルと数値リテラルに使用されます。 この構文は、文字列型から数値型または間隔型、数値型または間隔型から文字列型への変換を実行する場合に、ドライバー の作成者を支援するためにここに用意されています。 詳細については、「付録 D: データ型」の[「間隔リテラル](../../../odbc/reference/appendixes/interval-literals.md)と[数値リテラル](../../../odbc/reference/appendixes/numeric-literals.md)」を参照してください。
