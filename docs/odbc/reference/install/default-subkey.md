# <a name="default-subkey"></a>既定のサブキー
レジストリは、Default サブキーを使用して既定のデータ ソースを指定できます。 このサブキーは、データ ソース仕様サブキーの特殊なケースであり、他のデータ ソース仕様サブキーと同じ値を持ちます。 唯一の違いは、ODBC データ ソース サブキーの下に値として表示されないことです。
