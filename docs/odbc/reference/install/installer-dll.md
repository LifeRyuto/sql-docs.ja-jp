# <a name="installer-dll"></a>インストーラー DLL
> [!NOTE]  
>  WINDOWS XP および Windows Server 2003 以降では、ODBC が Windows のオペレーション システムに含まれています。 ODBC は、以前のバージョンの Windows にのみ明示的にインストールしてください。  
  
 インストーラー DLL には、ODBC コンポーネントのインストールと削除、コンポーネントに関するレジストリ情報の管理、およびデータ ソースに関するレジストリ情報の管理を行う関数が含まれています。 これは、マイクロソフトによって書かられており、再配布することができます。 インストーラ DLL の関数の詳細については、「[インストーラ DLL API リファレンス](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)」を参照してください。
