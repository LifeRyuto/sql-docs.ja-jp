# <a name="machine-data-sources"></a>マシン データ ソース
*マシンデータソース*は、ユーザ定義の名前でシステムに保存されます。 データ ソース名に関連付けられているすべての情報ドライバー マネージャーとドライバーがデータ ソースに接続する必要があります。 Xbase データ ソースの場合、Xbase ドライバの名前、Xbase ファイルを含むディレクトリの完全パス、およびシングルユーザー モードや読み取り専用など、これらのファイルの使用方法をドライバに伝えるオプションがあります。 Oracle データ ソースの場合、Oracle ドライバの名前、Oracle DBMS が存在するサーバ、使用する SQL Net ドライバを識別する SQL*Net\*接続文字列、およびサーバ上のデータベースのシステム ID が含まれます。
