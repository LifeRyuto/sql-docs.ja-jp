# <a name="standard-data-stream-protocol"></a>標準のデータ ストリーム プロトコル
標準データ ストリーム プロトコルは、異種 DBMS のデータにアクセスする 1 つの方法です。 実際、標準のデータ ストリーム プロトコルは既に存在します。

- ANSI/ISO リモート データベース アクセス (RDA) 標準: [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). ANSI/ISOシステムは約束を示していますが、現在では広く実装されていません。
