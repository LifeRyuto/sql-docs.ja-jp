# <a name="general-error-checks"></a>一般的なエラー チェック
ドライバー マネージャーは、1 つの一般的なエラーを確認します。 次のエラーが発生したときに常に SQL_ERROR を返します。関数は、ドライバーでサポートする必要があります。
