# <a name="transactions-in-odbc-odbc"></a>ODBC のトランザクション (ODBC)
ODBC のトランザクションは接続レベルで完了します。つまり、アプリケーションはトランザクションを完了すると、その接続のすべてのステートメント ハンドルを通じて実行されたすべての作業をコミットまたはロールバックします。
