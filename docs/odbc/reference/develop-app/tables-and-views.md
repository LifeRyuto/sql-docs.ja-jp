# <a name="tables-and-views"></a>[テーブルとビュー]
ODBC 関数では、テーブルとビューを交換できます。 用語*テーブル*はテーブルとビューの両方に使用されますが、用語*ビュー*が明示的に使用されている場合を除きます。
