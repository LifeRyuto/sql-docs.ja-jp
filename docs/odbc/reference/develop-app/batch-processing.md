# <a name="batch-processing"></a>バッチ処理
SQL ステートメントのバッチのサポートは広く普及していないので、相互運用可能なアプリケーションは、条件付きで使用するか、まったく使用しない必要があります。 詳細については、「[バッチの実行](../../../odbc/reference/develop-app/executing-batches.md)」を参照してください。
