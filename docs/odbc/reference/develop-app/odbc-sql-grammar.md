# <a name="odbc-sql-grammar"></a>ODBC SQL 文法
相互運用可能なアプリケーションでは、SQL ステートメントで ODBC SQL 文法を常に使用する必要があります。 しかし、この文法の中でもかなりのバリエーションが可能です。 詳細については、「 [SQL ステートメントの相互運用性](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md)」を参照してください。
