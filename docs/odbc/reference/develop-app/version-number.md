# <a name="version-number"></a>バージョン番号
ODBC には複数のバージョンがあり、それぞれ機能が異なります。 アプリケーションは、SQL_ODBC_VERオプションとSQL_DRIVER_ODBC_VERオプションを指定して**SQLGetInfo**を呼び出すことによって、ドライバー マネージャーと特定のドライバーサポートの ODBC バージョンを決定します。
