# <a name="diagnostic-tools"></a>診断ツール
2 つの機能を使用して、ODBC アプリケーションの問題を診断できます。 どちらも ODBC アドミニストレータから入手でき、ドライバー マネージャーによって実装されます。 *トレース*は、関数呼び出しのシーケンスをログ ファイルに記録する手段を提供します。 Visual Studio アナライザーを使用すると、分散環境でのコンポーネントの相互作用に関する情報を分析できます。
