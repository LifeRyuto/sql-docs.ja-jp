# <a name="connecting-with-sqldriverconnect"></a>SQLDriverConnect による接続
**接続文字列**を使用してデータ ソースに接続するために使用されます。 次**の**理由により **、SQLConnect**の代わりに接続が使用されます。  
  
-   アプリケーションでドライバ固有の接続情報を使用できるようにする。  
  
-   ドライバーがユーザーに対して接続情報を要求する場合  
  
-   データ ソースを指定せずに接続すること。
