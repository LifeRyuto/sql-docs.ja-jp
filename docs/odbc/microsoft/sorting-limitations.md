# <a name="sorting-limitations"></a>制限事項を並べ替え
GROUP BY 句、ORDER BY 句、SELECT DISTINCT ステートメント、または外部結合の中のソートキーの最大長は 255 バイトです。ソート行内のすべてのソート・キーの最大長は 65,500 バイトです。
