# <a name="supported-data-types-visual-foxpro-odbc-driver"></a>サポートされるデータ型 (Visual FoxPro ODBC ドライバー)
ドライバーでサポートされているデータ型の一覧は、ODBC API と Microsoft クエリで表示されます。  
  
## <a name="data-types-in-c-applications"></a>C アプリケーションのデータ型  
 C または C++ アプリケーションで[SQLGetTypeInfo](../../odbc/microsoft/sqlgettypeinfo-visual-foxpro-odbc-driver.md)関数を使用して、Visual FoxPro ODBC ドライバーでサポートされているデータ型の一覧を取得できます。  
  
## <a name="data-types-in-applications-using-microsoft-query"></a>Microsoft クエリを使用するアプリケーションのデータ型  
 アプリケーションで Microsoft クエリを使用して Visual FoxPro データ ソースに新しいテーブルを作成する場合は、[**新しいテーブル定義**] ダイアログ ボックスが表示されます。 [**フィールドの説明**] の [**種類]** ボックスには、単一の文字で表される[Visual FoxPro フィールド データ型](../../odbc/microsoft/visual-foxpro-field-data-types.md)が一覧表示されます。
