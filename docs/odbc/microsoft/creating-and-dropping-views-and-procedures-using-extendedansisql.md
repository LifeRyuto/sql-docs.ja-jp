# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>ExtendedAnsiSQL を使用したビューとプロシージャの作成と削除
ExtendedAnsiSQL フラグがオンになっている場合、Jet 4.0 は SQL を使用してビューとプロシージャを作成および削除する機能をサポートします。 これらのビューとプロシージャは、Access では表示されません。
