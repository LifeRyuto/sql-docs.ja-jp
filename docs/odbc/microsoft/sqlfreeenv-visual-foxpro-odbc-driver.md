# <a name="sqlfreeenv-visual-foxpro-odbc-driver"></a>SQLFreeEnv (Visual FoxPro ODBC ドライバー)
> [!NOTE]  
>  このトピックには、ビジュアル フォックス プロ ODBC ドライバー固有の情報が含まれています。 この関数の一般的な情報については[、ODBC API リファレンス](../../odbc/reference/syntax/odbc-api-reference.md)の該当するトピックを参照してください。  
  
 サポート: フル  
  
 ODBC API 準拠: コア レベル  
  
 Visual FoxPro ODBC ドライバーを閉じ、ドライバーに関連付けられているすべてのメモリを解放します。  
  
 詳細については *、ODBC プログラマ リファレンス*の[SQLFreeEnv](../../odbc/reference/syntax/sqlfreeenv-function.md)を参照してください。
