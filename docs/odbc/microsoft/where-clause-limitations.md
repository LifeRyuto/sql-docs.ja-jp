# <a name="where-clause-limitations"></a>WHERE 句の制限事項
WHERE 句の句の最大数は 40 です。  
  
 LONGVARBINARY 列および LONGVARCHAR 列は、最大 255 文字の長さのリテラルと比較できますが、パラメーターを使用して比較することはできません。
