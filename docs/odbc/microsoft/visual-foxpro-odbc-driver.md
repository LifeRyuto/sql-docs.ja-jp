# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC ドライバー
マイクロソフトビジュアルフォックスプロは、データベース構築とアプリケーション開発のための強力なオブジェクト指向環境です。 Microsoft Visual FoxPro ODBC ドライバーを使用すると、アプリケーションはオープン データベース接続 (ODBC) インターフェイスを使用して、Visual FoxPro および以前のバージョンの FoxPro のデータを開く、クエリ、および更新できます。  
  
 たとえば、次の操作を実行できます。  
  
-   クエリを使用して、Excel ワークシートからビジュアル FoxPro データをクエリおよび更新します。  
  
-   Word でビジュアル フォックスプロ データを使用して差し込み印刷のレターを作成します。  
  
-   クエリを実行し、Access からビジュアル フォックスプロ ビューとテーブルを更新します。  
  
-   Visual FoxPro を使用して、Visual Basic、Visual C++、および C アプリケーションのデータ ストアとして使用します。  
  
 このドライバを使用して、他の多くのタスクを実行できます。 次の表に、開始に役立ついくつかのトピックを示します。
