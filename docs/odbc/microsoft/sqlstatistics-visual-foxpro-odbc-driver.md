# <a name="sqlstatistics-visual-foxpro-odbc-driver"></a>SQLStatistics (Visual FoxPro ODBC ドライバー)
> [!NOTE]  
>  このトピックには、ビジュアル フォックス プロ ODBC ドライバー固有の情報が含まれています。 この関数の一般的な情報については[、ODBC API リファレンス](../../odbc/reference/syntax/odbc-api-reference.md)の該当するトピックを参照してください。  
  
 サポート: フル  
  
 ODBC API 準拠: レベル 1  
  
 単一のテーブルと、そのテーブルに関連付けられているインデックスまたはタグ名に関する統計情報のリストを取得します。 ドライバーは、結果セットとして情報を返します。  
  
 詳細については *、『ODBC プログラマ リファレンス*』の[「SQLStatistics」](../../odbc/reference/syntax/sqlstatistics-function.md)を参照してください。
