# <a name="date-arithmetic"></a>日付の算術演算子
ドライバーは、加算と減算の日付、時刻、またはタイムスタンプ列からの整数をサポートします。 日付の列の整数に加算または減算する日数を指定します。 時刻、またはタイムスタンプ列では、整数の加算または減算する秒数を指定します。
