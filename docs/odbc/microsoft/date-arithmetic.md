# <a name="date-arithmetic"></a>日付の算術演算子
ドライバーは、日付、時刻、または TIMESTAMP 列から整数を加算および減算をサポートします。 DATE 列の場合、整数は加算または減算する日数を指定します。 TIME または TIMESTAMP 列の場合、整数は加算または減算する秒数を指定します。
