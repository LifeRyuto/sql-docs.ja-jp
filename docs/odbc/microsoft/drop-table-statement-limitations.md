# <a name="drop-table-statement-limitations"></a>DROP TABLE ステートメントの制限事項
Microsoft Excel 5.0、7.0、または 97 ドライバを使用すると、DROP TABLE ステートメントによってワークシートが消去されますが、ワークシート名は削除されません。 ワークシート名はブックに残るため、同じ名前の別のワークシートを作成できません。
