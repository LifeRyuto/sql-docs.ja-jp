# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>ブックマーク サポート (Visual FoxPro ODBC ドライバー)
Visual FoxPro ODBC ドライバーは、単純なブックマークをサポートしています。 SQL_BOOKMARK_PERSISTENCE *InfoType*で[SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md)を呼び出すと、戻り値は SQL_BP_SCROLL になります。  
  
 ブックマークの詳細については、「[ブックマーク (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md)」を参照してください。
